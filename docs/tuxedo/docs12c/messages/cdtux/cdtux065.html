CMDTUX Messages 6500-6599

 

6500

INFO: WLS security realm name already specified, replace it with new realm name <realm-name>

Description

This is informational. The LAUTHSVR configuration file allows user to specify which WLS security realm is active in WLS Domain. The configuration file contains more than one entries for the WLS security realm name. The security realm name specified later will be used.

Action

Remove the duplicated entry from the LAUTHSVR configuration file.

See Also
