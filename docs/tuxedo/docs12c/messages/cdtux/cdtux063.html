CMDTUX Messages 6300-6399

 

6301

ERROR: A TLOGDEVICE cannot be configured if the NO_XA option is set

Description

The identifier for TLOGDEVICE as specified in the MACHINES section of the UBBCONFIG can not be set if the OPTIONS in the RESOURCE section has the NO_XA option set.

Action

Correct the incompatible value of the UBBCONFIG file.

See Also

ubbconfig(5) and tmloadcf (1)

6320

ERROR: Can't tpadmcall buffer buf

Description

ud was unable to send a buffer to tpadmcall to process. Additional information is printed indicating the cause of the failure.

Action

Check the reason for the failure. No action required.

See Also

ud(1), TM_MIB(5)

6323

ERROR: Unlicensed System Binary

Description

A valid license file has not been installed in $TUXDIR/udataobj/lic.txt. This license file must have a section for Engine with the specific version number..

Action

Buy a license and install the provided file in udataobj/lic.txt.

6331

ERROR: Memory allocation failure

Description

The system ran out of memory in the BRIDGE or BSBRIDGE

Action

Ensure that the operating system parameters are set correctly for the amount of memory on the machine and the amount of memory that can be used by a process. Reduce the memory usage on the machine or increase the amount of physical memory on the machine.

6332

ERROR: Error sending keepalive packet

Description

While sending the keepalive packet for BRIDGE communication, an error occurred.

Action

Check the network on the machine with the affected BRIDGE or BSBRIDGE.
