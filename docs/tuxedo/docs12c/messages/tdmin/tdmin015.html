TMADMIN Messages 1500-1599

 

1501

ERROR: Unlicensed System Binary

Description

A valid license file has not been installed in $TUXDIR/udataobj/lic.txt. This license file must have a section for Engine with the specific version number..

Action

Buy a license and install the provided file in udataobj/lic.txt.

1502

ERROR: Expired System Binary

Description

Your copy of the BEA WLE system software is an evaluation copy, and the expiration date has passed.

Action

If you wish to continue using the BEA WLE system, contact the company from which you received the product.

1507

ERROR: Machine required when specifying default object ID.

Description

When specifying a default value for object id in tmadmin the machine id default value must also be specified.

Action

Set the machine id default value prior to or during the assignment of a default object id value.

See Also

tmadmin(1)

1508

ERROR: Interface exists in multiple groups, must specify group name.

Description

While attempting to change the load, priority or timeout value of an interface the interface name was not fully qualified with a group name.

Action

Retry the operation with the group name specified.

See Also

tmadmin(1)

1509

ERROR: No such routing name.

Description

An attempt was made to invoke the tmadmin command printroute with an invalid routing name.

Action

Verify the correct routing name and retry the operation, or, invoke the printroute without a routing name argument.

See Also

tmadmin(1)

1558

ERROR: No such JDBC Connection Pool

Description

An attempt was made to invoke the tmadmin command printjdbcconnpool with an invalid JDBC connection pool name.

Action

Verify the correct JDBC connection pool name and retry the operation, or, invoke the printjdbcconnpool without any argument to get a list of available JDBC connection pools.

See Also

tmadmin(1)

1599

ERROR: The new value is not within the range of min-max

Description

This message is printed when an illegal value has been given to one of the changeload, changepriority, or changetrantime commands of tmadmin(1). Interface loads must be between 1 and 32767, inclusive; service priorities must be between 1 and 100, inclusive; and transaction times must be between 0 and 2,147,483,647, inclusive.

Action

Re-enter the command using a value within the appropriate range. The SERVICES section of the ubbconfig(5) manual page describes the correct ranges and defaults for these service attributes.

See Also

ubbconfig(5)
