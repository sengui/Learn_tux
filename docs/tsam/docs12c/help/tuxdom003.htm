Viewing Tuxedo ULOG

You can view log messages occurred recently or in a defined time period. All the messages are listed in the log page by default. You can select to group them by message ID or Type.

  1. From the Tuxedo Domain action bar, click Log.

  2. In the Credentials page that appears, choose a Credential according to the screen prompts and click OK.


Note:

  • Up to 1000 log messages can be searched out at a time. You may need to narrow down the search time period if the result exceeds the maximum limit.

  • Auto refresh is available only when you display the recent log messages.

  • Both master and slave machines support showing ULOG. Slave machine ULOG is fetched from tlisten component running in the master machine.

  • At machine level, only master machine's log menu could be invoked.
