Monitoring a Tuxedo Domain

Use the Tuxedo Domain page to view general information about the server domain associated with this farm. Click on any of the server instances within the domain to view detailed information.

The following details are displayed for Tuxedo domain target:

  • Summary

    Shows two kinds of information: general properties and statistics.

    • General properties

      Properties Description
      Name The domain name in the format of [DOMAINID]:[PMID of the master machine]: [PMID of the backup machine]:[IPCKEY]

      Note: When making swap operation, the name is not changed.
      DOMAINID DOMAINID parameter in TUXCONFIG
      IPCKEY IPCKEY parameter in TUXCONFIG
      MASTER The LMIDs list of master Tuxedo machines
      MODEL MODEL parameter in TUXCONFIG
      SECURITY SECURITY parameter in TUXCONFIG
      OPTIONS OPTIONS parameter in TUXCONFIG

    • Statistics metrics

      Metrics Description
      tpconnect Number The invoking times of tpconnect method per minute in the Tuxedo domain during the interval
      tpdequeue Number The invoking times of tpdequeue method per minute in the Tuxedo domain during the interval.
      tpenqueue Number The invoking times of tpenqueue method per minute in the Tuxedo domain during the interval.
      tppost Number The invoking times of tppost method per minute in the Tuxedo domain during the interval.
      tpcall and tpacall Number The invoking times of tpcall method per minute in the Tuxedo domain during the interval.
      tpsubscribe Number The invoking times of tpsubscribe method per minute in the Tuxedo domain during the interval.
      Number of transactions initiated The number of XA transactions initiated per minute in the Tuxedo domain during the interval.
      Number of transactions aborted The number of XA transactions aborted per minute in the Tuxedo domain during the interval.
      Number of transactions committed The number of XA transactions committed per minute in the Tuxedo domain during the interval.
      Completed Workload The number of XA transactions aborted or committed per minute in the Tuxedo domain during the interval.
      Request Number The invoking times per minute of tpcall, tpacall and tpconnect methods in the Tuxedo domain during the interval.
      Complete Transaction Number The work load per minute completed in the Tuxedo domain during the interval.

  • Performance Metric Chart

    Represents domain status. On the vertical axis, 1 represents available, and 0 represents unavailable.

    You can select to view the real-time statistics data in line chart or table.

  • Configuration Pane

    Shows UBBCONFIG fields under this target. For more information, see Tuxedo Domain in ECM Table section in Enterprise Manager for Oracle Tuxedo Reference Guide.
