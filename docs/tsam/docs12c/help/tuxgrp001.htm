Monitoring a Tuxedo Group

The Tuxedo group page shows the following information:

  • General properties

    • Name: Tuxedo group name in the format of [Domain Name]/[MasterLMID]/[SRVGRP].

    • LMID: The current LMID, this property is especially necessary for a group related to two Tuxedo machines.

    • SRVGRP: Group name.

    • GRPNO: Tuxedo installation directory.

  • Configuration Pane

    Shows UBBCONFIG fields under this target. For more information, see Tuxedo Group in ECM Table section in Enterprise Manager for Oracle Tuxedo Reference Guide.
