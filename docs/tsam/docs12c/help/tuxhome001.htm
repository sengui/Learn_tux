Monitoring a Tuxedo Home

The Tuxedo Home page shows the following general information:

  • Name: Tuxedo home name in the format of [Host Name]:[TUXDIR].

  • Version: Tuxedo installation release number.

  • RP Level: Rolling patch package level.

  • Directory: Tuxedo installation directory.
