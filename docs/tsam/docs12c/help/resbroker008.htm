Configuring CICS Configuration

The CICS Configuration page consists of three areas:

  • CICS Configuration panel, from which you can save or cancel all the current configurations made on the page.

  • Package Available table, which lists all available CICS packages. Here packages can be added to the Package Selected table by selecting the desired items checkbox and clicking Add.

  • Pakcage Selected table, which lists all selected CICS packages. From Show list at the top of panel, you can switch to show the selected CICS package between two views:

    • Files: The Package view, which lists package information. Clicking on Remove can remove all the selected packages from the table.

    • Merging: The Configuration File view, which lists the merged same type configuration file from all selected CICS packages.

Each package may contain a set of resource files. During process of adding each package to the domain, the .desc files with the same name are merged into one file, which is shown in the Configuration File view.

A confliction validation is performed on the merged file. Once a confliction is detected, an error message is shown and the confliction content is highlighted in Red color. Clicking the relative button can show the confliction details.


Note:

The package assembling status can only be saved when there is no confliction.
