Monitoring a Tuxedo Server

The following details are displayed for Tuxedo server target:

  • Summary

    Shows two kinds of information: general properties and statistics.

    • General properties

      Properties Description
      Name The Tuxdeo server name in the format of [Group Name]/[AOUT]:[ SRVID]
      SRVGRP Group name
      SRVID SRVID parameter for the server entry defined in SERVERS section of UBBCONFIG(5)
      AOUT Process name

    • Statistics metrics

      Metrics Description
      Average Service Execution Time The average execution time in microsecond aggregated by the service name.
      Average Service CPU Time The average CPU occupancy time in microsecond aggregated by the service name.
      Service Success Number The successful request number per minute aggregated by the service name.
      Service System Failure Number The request number of system failure per minute (tperrno is not TPESVCFAIL) aggregated by the service name.
      Service User Failure Number The request number of user failure per minute (tperrno is TPESVCFAIL) aggregated by the service name.
      Request Number The request number per minute aggregated by the service name.
      Service Success Ratio The successful ratio of service requests on the Tuxedo server target.

  • Performance Metric Chart

    Represents request number per minute.

    You can select to view the real-time statistics data in line chart or table.

  • Service

    For information about service metrics, see Service Metrics in Tuxedo Server section in Enterprise Manager for Oracle Tuxedo Reference Guide.

  • IPC Queue

    For information about IPC queue metrics, see IPC Queue Metrics in Tuxedo Server section in Enterprise Manager for Oracle Tuxedo Reference Guide.

  • Configuration Pane

    Shows UBBCONFIG fields under this target. For more information, see Tuxedo Domain in ECM Table section in Enterprise Manager for Oracle Tuxedo Reference Guide.

You can suspend or resume the selected services in the current server from this page.
