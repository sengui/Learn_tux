Aborting Transactions

You can stop the selected transactions before it is complete.

  1. From the Tuxedo Machine action bar, click Transactions.

  2. In the Credentials page that appears, choose a Credential according to the screen prompts and click OK.

  3. In the Transaction page, click Abort.

    Any data that changed as a result of that transaction is rolled back to previous value at the start of the transaction. The transaction must be in either the ACTIVE, ABORTONLY, or COMCALLED state.


Caution:

There is a very slight risk that the data roll back may not be successful. You can minimize the risk by aborting a transaction early in the process.
