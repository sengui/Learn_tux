Creating a Queue

You can suspend or resume the selected services in the current server.

  1. From the Tuxedo OTMQ action bar, click Create Queue.

  2. Enter the queue parameters in the page that appears and click Create Queue.

    A creation result page appears, showing either the congratulation message or the error message upon failure.
