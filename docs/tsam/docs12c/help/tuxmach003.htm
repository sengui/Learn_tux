Detecting Dead Clients or Servers

You can detect dead clients or servers on the current machine.

  1. From the Tuxedo Machine action bar, click Clean.

  2. In the Confirmation prompt that appears, click Clean.

If the machine has been partitioned from the application MASTER site, global bulletin board entries for the machine are removed. This combination is allowed when the application is in the ACTive state and the T_MACHINE object is in either ACTive or PARtitioned state. Successful return for a non-partitioned machine leaves the state unchanged. Successful return for a partitioned machine leaves the object in the INActive state.
