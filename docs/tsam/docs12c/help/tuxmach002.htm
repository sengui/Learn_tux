Migrating Servers Between Two Machines

You can migrate the server which is associated with two Tuxedo machines from the current machine to another.

  1. From the Tuxedo Machine action bar, click Migrate.

  2. In the Credentials page that appears, choose a Credential according to the screen prompts and click OK.

The process of migration can be monitored by Enterprise Manager. You can choose to run the process in backend and stop the migration process.


Note:

The group that includes LMS can not be migrated. Please make sure the group that includes LMS does not have secondary LMID.
