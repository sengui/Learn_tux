Monitoring a Tuxedo LMS

The Tuxedo LMS page shows the following general information:

  • Name: Tuxedo tlisten name in the format of [Group Name]/[AOUT]:[ SRVID].

  • SRVGRP: Group name.

  • SRVID: SRVID parameter for the server entry defined in SERVERS section of UBBCONFIG(5).

  • AOUT: process name.

  • TUXCONFIG: The tuxconfig path of Tuxedo LMS.
