Starting Up a Target

In most target home pages, you can activate the current target as well as all the targets belonging to it by clicking the Start Up button on the top action bar. For some targets, e.g. Tuxedo domain, it provides specific options:

  • Only startup Administrative Servers.

  • Only startup Application Servers.
