Monitoring a Tuxedo Tlisten

The Tuxedo Tlisten page shows the following general information:

  • Name: Tuxedo tlisten name in the format of [nlsaddr]/tlisten.

  • Device: Full pathname of the network device.

  • uid

  • sec_principal_name

  • sec_principal_location

  • sec_principal_passvar

  • Zbits

  • nlsaddr

  • jmxaddr: The address of RMI connector of embedded JMX agent.
