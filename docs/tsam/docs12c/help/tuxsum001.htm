Viewing Tuxedo Summary

The Tuxedo Summary page filters all Tuxedo domain targets from a large number of Enterprise Manager monitoring targets, allowing you to centrally manage tuxedo targets and easily navigate to each target home page.

You can perform operations on the page by the following action buttons:

  • Add: Adds Tuxedo domains. You can choose one of the following:

    • Discover the existing domains. How?

    • Create a new Tuxedo domain. How?

  • Edit: Edits the highlighted target.

  • Remove: Removes the Tuxedo target and all metrics associated with it from Enterprise Repository. For Tuxedo domain, this action deletes all tuxedo targets belonging to it.

    A Tuxedo tlisten target can be removed only if none of Tuxedo Machine targets has "Monitored_by" associated, otherwise a warning message pops up to prevent you from proceeding.

  • Configure: Configures the properties of the highlighted target, such as version, RP level, directory, and word size.

  • Manage Application Package: Uploads or deletes application packages that are specially used for resource broker feature. More

Search Panel

You can quickly search for a target by specific fields using the Search panel.

  1. Select the field to be searched in the drop-down list.

  2. Enter an appropriate search string in the text box to run a standard search.

You can also use Advanced Search to run the search with further filtering.
