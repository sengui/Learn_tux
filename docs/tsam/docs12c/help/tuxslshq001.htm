Monitoring a Tuxedo Queue Server

The Tuxedo /Q page shows the following information:

  • General properties

    • Name: Target name in the format of [Group Name]/[AOUT]:[ SRVID].

    • Group Name: Name of the group the target belongs to.

    • SRVID: SRVID parameter for the server entry defined in SERVERS section of UBBCONFIG(5).

    • Queue Space Name: Qspace name.

    • Queue Space Path: Absolute pathname of the file or device where the application queue space is located in.

  • Performance Metric Chart

    Presents one kind data of queue space statistics metrics:

    • Total pending message number

    You can select to view the real-time statistics data in line chart or table.

  • Queue Statistics Pane

    Shows the data defined in queue statistics metrics. For more information, see Queue Statistics Metrics in Tuxedo Targets section in Enterprise Manager for Oracle Tuxedo Reference Guide.
