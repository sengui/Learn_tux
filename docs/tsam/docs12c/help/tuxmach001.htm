Monitoring a Tuxedo Machine

Use this page to view information about the selected Oracle Tuxedo Machine instance and applications running within it. Click on an application for detailed information about its current state.

The following regions are displayed for Tuxedo Machine target:

  • Summary

    Shows two kinds of information: general properties and statistics.

    • General properties

      Properties Description
      Name The machine name in the format of [Domain Name]/[LMID]
      PMID Physical machine ID
      LMID Logic machine name
      APPDIR
      ENVFILE
      TUXCONFIG

    • Statistics metrics

      Metrics Description
      tpconnect Number The invoking times of tpconnect method per minute in Tuxedo machine during the interval.
      tpdequeue Number The invoking times of tpdequeue method per minute in Tuxedo machine during the interval.
      tpenqueue Number The invoking times of tpenqueue method per minute in Tuxedo machine during the interval.
      tppost Number The invoking times of tppost method per minute in Tuxedo machine during the interval.
      tpcall and tpacall Number The invoking times of tpcall method per minute in the Tuxedo machine during the interval.
      tpsubscribe Number The invoking times of tpsubscribe method per minute in the Tuxedo machine during the interval.
      Number of transactions initiated The number of XA transactions initiated per minute in the Tuxedo machine during the interval.
      Number of transactions committed The number of XA transactions committed per minute in the Tuxedo machine during the interval.
      Number of transactions aborted The number of XA transactions aborted per minute in the Tuxedo machine during the interval.
      Complete Transaction Number The work load per minute completed in the Tuxedo machine during the interval.
      Request Number The invoking times per minute of tpcall, tpacall and tpconnect methods in the Tuxedo machine during the interval.
      Completed Workload The number of XA transactions aborted or committed per minute in the Tuxedo machine during the interval.

  • Performance Metric Chart

    Represents request number per minute.

    You can select to view the real-time statistics data in line chart or table.

  • Configuration Pane

    Shows UBBCONFIG fields under this target. For more information, see Tuxedo Domain in ECM Table section in Enterprise Manager for Oracle Tuxedo Reference Guide.
