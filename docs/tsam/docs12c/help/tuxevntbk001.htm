Monitoring a Tuxedo Event Broker

The Tuxedo group page shows the following information:

  • General properties

    • Name: Tuxedo user or system event reporting server name in the format of [Group Name]/[AOUT]:[ SRVID].

    • SRVGRP: Group name.

    • SRVID: SRVID parameter for the server entry defined in SERVERS section of UBBCONFIG(5).

    • AOUT: Process name.

  • Performance Metric Chart

    Represents one kind data of Tuxedo Event Broker metrics:

    • Notified Event Number per minute

    You can select to view the real-time statistics data in line chart or table.
