Monitoring a Tuxedo Workstation Listener

The following details are displayed for Tuxedo Workstation listener target:

  • Summary

    Shows two kinds of information: general properties and statistics.

    • General properties

      Properties Description
      Name The Tuxedo workstation listener name in the format of [Group Name]/[AOUT]:[ SRVID]
      SRVGRP Group name
      SRVID SRVID parameter for the server entry defined in SERVERS section of UBBCONFIG.
      AOUT Process name

    • Statistics metrics

      Metrics Description
      Active Handler Number The number of the active handlers spawned by the listener.
      Client Number The number of the clients that are connecting with the handlers spawned by the listener.
      Request Number (per minute) The number of the requests that are sent to the handlers spawned by the listener per minute.
      Outstanding Requests Number The number of the requests that are waiting for the reply on the handlers spawned by the listener.

  • Performance Metric Chart

    Represents two kinds data of listener metrics:

    • Client Number

    • Request number per minute

    You can select to view the real-time statistics data in line chart or table.

You can view all the clients under a handler or kill the selected handlers from this page.
