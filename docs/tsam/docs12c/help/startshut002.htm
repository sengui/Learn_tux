Shutting Down a Target

In most target home pages, you can deactivate the current target and all the targets belonging to it by clicking the Shut Down button on the top action bar. For some targets, e.g. Tuxedo domain, it provides specific options:

  • Only deactivate Administrative Servers

  • Only deactivate Application Servers

  • Delay: Delay the shutdown for the period specified in the textbox

  • Signal:

    • None: No signal is specified and use system default way to shut down

    • TERM: use SIGTERM signal to shut down

    • KILL: use SIGKILL signal to shut down

  • Force deactive even if clients remain
