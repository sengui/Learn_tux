Overview

The Oracle TSAM Plus Agent framework collects the performance metrics when Oracle TSAM Plus is enabled. The framework covers the major performance sensitive areas in Tuxedo applications, that is call path stages, services, transactions and system servers. Oracle TSAM Plus Agent uses Oracle Tuxedo FML32 typed buffers to contain the metrics collected so that each metric is defined as a built-in FML32 field. The monitoring points depend on the monitoring types and only apply to Oracle Tuxedo ATMI applications. Table 2-1 lists the call path monitoring points.
