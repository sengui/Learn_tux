CMDSCA Messages 1000-1099

 

1051

ERROR: Command line option "-m" contains characters other than numeric digits.

Description

The -m command line option should only contain numeric digits.

Action

Use numeric digits only.

1052

ERROR: Command line option "-m" argument is too long.

Description

The maximum value for the "-m" argument is 1024.

Action

Use a "-m" argument lower than or equal to 1024.

1053

ERROR: Command line option "-m" argument is > 1024.

Description

The maximum value for the "-m" argument is 1024.

Action

Use a "-m" argument lower than or equal to 1024.

1054

ERROR: Unknown command line option option is found.

Description

The command line option indicated is not valid.

Action

Revise the command syntax.

1055

ERROR: Command line option "-t" can only be specified once.

Description

The "-t" option has been specified more than once on the command line.

Action

Revise the command line entered.

1056

ERROR: The specified input file filename either does not exist or does not have read permission.

Description

The command could not find the specified file, or the file could not be accessed.

Action

Verify that the file exists and has appropriate permissions, then re-run the command.

1057

WARN: "-Gi" and "-Gs" options are not applicable to Java and will be ignored.

Description

These options have no effect when generating Java code.

Action

No action required.

1058

WARN: Nothing to generate, stop processing!

Description

The command has been invoked to generate a Java interface, but but the "-ga" option has not been specified.

Action

Revise the command syntax and re-run it.

1059

ERROR: The specified service root directory directory is not valid.

Description

The directory specified with the "-s" option is not a directory or does not exist.

Action

Revise the command syntax and re-run it.

1060

ERROR: No record matching search criteria found.

Description

No service matching the search criteria could be found in the input.

Action

Revise the command syntax and re-run it.

1061

ERROR: Custom buffer types are not supported.

Description

Action

No action required.
