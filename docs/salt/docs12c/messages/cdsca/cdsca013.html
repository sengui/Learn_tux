CMDSCA Messages 1300-1399

 

1300

ERROR: Invalid option: opt

Description

User entered invalid command line option opt for scaadmin.

Action

Use the correct command line option.

See Also

scaadmin

1301

Description

Invalid scaadmin command line argument.

Action

Remove the invalid command line argument.

See Also

scaadmin

1302

ERROR: TUXCONFIG environment variable not set.

Description

TUXCONFIG environment variable is not set.

Action

Set the TUXCONFIG environment variable to a valid TUXCONFIG file and re-run the scaadmin utility.

1303

ERROR: TUXCONFIG must be an absolute pathname.

Description

The TUXCONFIG environment variable must be of full path name.

Action

Set the TUXCONFIG environment variable to a valid TUXCONFIG file with full path name from system root, and then re-run the scaadmin utility.

1304

Description

Usage information of the program procname.

Action

No action required.

1305

ERROR: Error while obtaining the Bulletin Board parameters

Description

An error was produced in scaadmin while retrieving the Bulletin Board parameters. A more specific error explaining the situation should already exist in the user log.

Action

Check the user log for a more specific error message. Check that TUXCONFIG is readable, and re-run the scaadmin.

See Also

scaadmin

1306

ERROR: Could not determine level of authentication required.

Description

when scaadmin starts up, it calls tpchkauth(3c) to see what level of authentication is required by the application. This message is issued when tpchkauth(3c) fails. tpchkauth(3c) can fail if scaadmin cannot find the TUXCONFIG file.

Action

Make sure the TUXCONFIG environment variable is set correctly.

See Also

scaadmin

1307

ERROR: Could not allocate internal TPINIT typed buffer.

Description

Executing the scaadmin utility error. An error was encountered while trying to obtain virtual memory for the TPINIT buffer.

Action

Attempt to identify and stop any extraneous processes to free up virtual memory on your system, and then retry the utility.

See Also

scaadmin

1308

ERROR: Could not find entry for user UID.

Description

When scaadmin is invoked, the invoker;s login ID is used as the "User Name" when scaadmin joins the application. If there is an error retrieving the login ID from the system password file, this message is printed.

Action

Contact your system administrator to ensure that your password entry contains your user ID (UID) and your login.

See Also

scaadmin

Action

See Also

scaadmin

1310

ERROR: Unable to obtain application password.n

Description

If an applicatoion password is required, scaadmin either prompts for it when the invoker is attached to a terminal, or it reads from the environment variable, APP_PW, if standard input is not terminal. This message is issued when the password cannot be obtained. The most common cause for this is that APP_PW is not set and standard input is not a terminal.

Action

If scaadmin is not attached to a terminal when invoked then check that the environment variable, APP_PW, is set correctly.

See Also

scaadmin

1311

ERROR: can't become a client - tpinit() failed - n

Description

Execution of wsadmin failed because the user could not join the application. The printed string indicates the reason for the failure. This message can be printed either because no more clients can join the application or because the user failed security permissions.

Action

If the reason is that no more clients can join the application, re-try the operation a later time. The system should be configured to allow for more clients.

See Also

scaadmin
